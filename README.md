# TypeWord

한국 속담을 받아와 연습하는 타자연습 웹앱입니다.

## 사용 방법
1. 저장소를 GitHub에 올리고 `main` 브랜치로 푸시하세요.
2. `.github/workflows/pages.yml`가 포함되어 있으면 GitHub Actions가 Pages로 배포합니다.
3. 커스텀 도메인을 사용하려면 `CNAME` 파일에 도메인을 적으세요.

## 파일 구성
- index.html
- css/styles.css
- js/app.js
- assets/logo.svg
- .github/workflows/pages.yml
- CNAME
